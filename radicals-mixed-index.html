    Section 8.7 Radicals of Mixed Index

    Objective: Reduce the index on a radical and multiply or divide radicals of different index.
    Knowing that a radical has the same properties as exponents (written as a ratio) allows us to manipulate radicals in new ways. One thing we are allowed to do is reduce, not just the radicand, but the index as well. This is shown in the following example.

    Example 8.7.1.

    \begin{align*} \sqrt[8]{x^6y^2} \amp \quad \text{ Rewrite as rational exponent } \\ \left(x^6y^2\right)^{1/8} \amp\quad \text{ Multiply exponents } \\ x^{6/8}y^{2/8} \amp\quad \text{ Reduce each fraction } \\ x^{3/4}y^{1/4} \amp\quad \text{ All exponents have denominator of $4$, this is our new index } \\ \sqrt[4]{x^3y} \amp\quad \text{ Our Solution } \end{align*}
    What we have done is reduced our index by dividing the index and all the exponents by the same number (\(2\) in the previous example). If we notice a common factor in the index and all the exponnets on every factor we can reduce by dividing by that common factor. This is shown in the next example.

    Example 8.7.2.

    \begin{align*} \sqrt[24]{a^6b^9c^{15}} \amp \quad \text{ Index and all exponents are divisible by } 3 \\ \sqrt[8]{a^2b^3c^{5}} \amp\quad \text{ Our Solution } \end{align*}
    We can use the same process when there are coefficients in the problem. We will first write the coefficient as an exponential expression so we can divide the exponet by the common factor as well.

    Example 8.7.3.

    \begin{align*} \sqrt[9]{8m^6n^3} \amp \quad \text{ Write $8$ as $2^3$ } \\ \sqrt[9]{2^3m^6n^3} \amp\quad \text{ Index and all exponents are divisible by } 3 \\ x^{6/8}y^{2/8} \amp\quad \text{ Reduce each fraction } \\ \sqrt[3]{2m^2n} \amp\quad \text{ Our Solution } \end{align*}
    We can use a very similar idea to also multiply radicals where the index does not match. First we will consider an example using rational exponents, then identify the pattern we can use.

    Example 8.7.4.

    \begin{align*} \sqrt[3]{ab^3} \sqrt[4]{a^2b} \amp \quad \text{ Rewrite as rational exponents } \\ \left(ab^2\right)^{1/3}\left(a^2b\right)^{1/4} \amp\quad \text{ Multiply exponents } \\ a^{1/3}b^{2/3}a^{2/4}b^{1/4} \amp\quad \text{ To have one radical need a common denominator, } 12 \\ a^{\frac{4}{12}}b^{\frac{8}{12}}a^{\frac{6}{12}}b^{\frac{3}{12}} \amp\quad \text{ Write under a single radical with common index, } 12 \\ \sqrt[12]{a^4b^8a^6b^3} \amp\quad \text{ Add exponents } \\ \sqrt[12]{a^10b^11} \amp\quad \text{ Our Solution } \end{align*}
    To combine the radicals we need a common index (just like the common denominator). We will get a common index by multiplying each index and exponent by an integer that will allow us to build up to that desired index. This process is shown in the next example.

    Example 8.7.5.

    \begin{align*} \sqrt[4]{a^2b^3} \sqrt[6]{a^2b} \amp \quad \text{ Common index is } 12 \\ \amp\quad \text{ Multiply first index and exponents by $3$, second by } 2 \\ \sqrt[12]{a^6b^9a^4b^2} \amp\quad \text{ Add exponents } \\ \sqrt[12]{a^{10}b^{11}} \amp\quad \text{ Our Solution } \end{align*}
    Often after combining radicals of mixed index we will need to simplify the resulting radical.

    Example 8.7.6.

    \begin{align*} \sqrt[5]{x^3y^4} \sqrt[3]{x^2y} \amp \quad \text{ Common index is } 15 \\ \amp\quad \text{ Multiply first index and exponents by $3$, second by } 5 \\ \sqrt[15]{x^9y^{12}x^{10}y^5} \amp\quad \text{ Add exponents } \\ \sqrt[15]{x^{19}y^{17}} \amp\quad \text{ Simplify by dividing exponents by index, remainder is left inside } \\ xy\sqrt[15]{x^4y^2} \amp\quad \text{ Our Solution } \end{align*}
    Just as with reducing the index, we will rewrite coefficients as exponential expressions. This will aslo allow us to use exponent properties to simplify.

    Example 8.7.7.

    \begin{align*} \sqrt[3]{4x^2y} \sqrt[4]{8xy^3} \amp \quad \text{ Rewrite $4$ as $2^2$ and $8$ as } 2^3 \\ \sqrt[3]{2^2x^2y} \sqrt[4]{2^3xy^3} \amp \quad \text{ Common index is } 12 \\ \amp\quad \text{ Multiply first index and exponents by $4$, second by } 3 \\ \sqrt[12]{2^8x^8y^4 2^9x^3y^9} \amp\quad \text{ Add exponents (even on the $2$) } \\ \sqrt[12]{2^{17}x^{11}y^{13}} \amp\quad \text{ Simplify by dividing exponents by index, remainder is left inside } \\ 2y\sqrt[12]{2^5x^{11}y} \amp\quad \text{ Simplify } 2^5 \\ 2y\sqrt[12]{32x^{11}y} \amp\quad \text{ Our Solution } \end{align*}
    If there is a binomial in the radical then we need to keep that binomial together through the entire problem.

    Example 8.7.8.

    \begin{align*} \sqrt{3x(y+z)} \sqrt[3]{9x(y+z)^2} \amp \quad \text{ Rewrite $9$ as $3^2$ } \\ \sqrt{3x(y+z)} \sqrt[3]{3^2x(y+z)^2} \amp \quad \text{ Common index is } 6 \\ \amp\quad \text{ Multiply first index and exponents by $3$, second by } 2 \\ \sqrt[6]{3^3x^3(y+z)^3 3^4x^2(y+z)^4} \amp\quad \text{ Add exponents, keep $(y+z)$ as binomial } \\ \sqrt[6]{3^7x^5 (y+z)^7} \amp\quad \text{ Simplify by dividing exponents by index, remainder is left inside } \\ 3(y+z)\sqrt[6]{3x^5(y+z)} \amp\quad \text{ Our Solution } \end{align*}
    World View Note: Originally the radical was just a check mark with the rest of the radical expression in parenthesis. In \(1637\) Rene Descartes was the first to put a line over the entire radical expression
    The same process is used for dividing mixed index as with multiplying mixed index. The only difference is our final answer cannot have a radical over the denominator.

    Example 8.7.9.

    \begin{align*} \frac{\sqrt[6]{x^4y^3z^2}}{\sqrt[8]{x^7y^2z}} \amp \quad \text{ Common index is $24$ Multiply first group by $4$, second by } 3 \\ \sqrt[24]{\frac{x^{16}y^{12}z^8}{x^{21}y^6z^3}} \amp\quad \text{ Subtract exponents } \\ \sqrt[24]{x^{-5}y^6z^5} \amp\quad \text{ Negative exponent moves to denominator. } \\ \sqrt[24]{\frac{y^6z^5}{x^5}} \amp\quad \text{ Do not want denominator in radical, need $24$x's, ($19$ more).} \\ \sqrt[24]{\frac{y^6z^5}{x^5}\frac{x^{19}}{x^{19}}} \amp\quad \text{ Multiply numerator and denominator by } x^{19} \\ \sqrt[24]{\frac{x^{19}y^6z^5}{x^24}} \amp\quad \text{ Simplify denominator } \\ \frac{\sqrt[24]{x^{19}y^6z^5}}{x} \amp\quad \text{ Our Solution } \end{align*}
