    Section 10.5 Logarithmic Functions

    Objective: Convert between logarithms and exponents and use that relationship to solve basic logarithmic equations.
    The inverse of an exponential function is a new function known as a logarithm. Logarithms are studied in detail in advanced algebra, here we will take an introductory look at how logarithms work. When working with radicals we found that there were two ways to write radicals. The expression \(\displaystyle \sqrt[\leftroot{-3}\uproot{3}m]{a^n} \) could be written as \(\displaystyle a^{\frac{n}{m}}\text{.}\) Each form has its advantages; thus, we need to be comfortable using both the radical form and the rational exponent form. Similarly, an exponent can be written in two forms, each with its own advantages. The first form we are very familiar with, \(b^x = a\text{,}\) where \(b\) is the base, \(a\) can be thought of as our answer, and \(x\) is the exponent. The second way to write this is with a logarithm, \(\log_b a = x \text{.}\) The word “log” tells us that we are in this new form. The variables all still mean the same thing. \(b\) is still the base, \(a\) can still be thought of as our answer.
    Using this idea the problem \(5^2 = 25\) could also be written as \(\log_5 25 = 2 \text{.}\) Both mean the same thing, both are still the same exponent problem, but just as roots can be written in radical form or rational exponent form, both our forms have their own advantages. The most important thing to be comfortable doing with logarithms and exponents is to be able to switch back and forth between the two forms. This is what is shown in the next few examples.

    Example 10.5.1.

    Write each exponential equation in logarithmic form.
    \begin{align*} m^3=5 \amp \quad \text{ Identify base $m$, answer $5$, and exponent } 3 \\ \log_m 5=3 \amp\quad \text{ Our solution } \\ \\ 7^2=b \amp\quad \text{ Identify base $7$, answer $b$, and exponent } 2 \\ \log_7 b=2 \amp\quad \text{ Our solution } \\ \\ \left(\frac{2}{3}\right)^4=\frac{16}{81} \amp\quad \text{ Identify base $\frac{2}{3}$, answer $\frac{16}{81}$, and exponent } 4 \\ \log_{\frac{2}{3}} \frac{16}{81} = 4 \amp\quad \text{ Our Solution } \end{align*}

    Example 10.5.2.

    Write each logarithmic equation in exponential form
    \begin{align*} \log_4 16=2 \amp \quad \text{ Identify base $4$, answer $16$, and exponent } 2 \\ 4^2=16 \amp\quad \text{ Our solution } \\ \\ \log_3 x =7 \amp\quad \text{ Identify base $3$, answer $x$, and exponent } 7 \\ 3^7=x \amp\quad \text{ Our solution } \\ \\ \log_9 3=\frac{1}{2} \amp\quad \text{ Identify base $9$, answer $3$, and exponent } \frac{1}{2} \\ 9^{\frac{1}{2}}=3 \amp\quad \text{ Our Solution } \end{align*}
    Once we are comfortable switching between logarithmic and exponential form, we are able to evaluate and solve logarithmic expressions and equations. We will first evaluate logarithmic expressions. An easy way to evaluate a logarithm is to set the logarithm equal to \(x\) and change it into an exponential equation.

    Example 10.5.3.

    \begin{align*} \text{Evaluate} \log_2 64 \amp \quad \text{ Set logarithm equal to } x \\ \log_2 64=x \amp\quad \text{ Change to exponent form } \\ 2^x=64 \amp\quad \text{ Write as common base, } 64=2^6 \\ 2^x=2^6 \amp\quad \text{ Same base, set exponents equal } \\ x=6 \amp\quad \text{ Our Solution } \end{align*}

    Example 10.5.4.

    \begin{align*} \text{Evaluate} \log_{125} 5 \amp \quad \text{ Set logarithm equal to } x \\ \log_{125} 5=x \amp\quad \text{ Change to exponent form } \\ 125^x=5 \amp\quad \text{ Write as common base, } 125=5^3 \\ (5^3)^x=5 \amp\quad \text{ Multiply exponents } \\ 5^{3x}=5 \amp\quad \text{ Same base, set exponents equal } (5=5^1) \\ 3x=1 \amp\quad \text{ Solve } \\ \overline{\ 3\ } \quad \overline{\ 3\ } \amp\quad \text{ Divide both sides by } 3 \\ x=\frac{1}{3} \amp\quad \text{ Our Solution } \end{align*}

    Example 10.5.5.

    \begin{align*} \text{Evaluate} \log_3 \frac{1}{27} \amp \quad \text{ Set logarithm equal to } x \\ \log_3 \frac{1}{27}=x \amp\quad \text{ Change to exponent form } \\ 3^x=\frac{1}{27} \amp\quad \text{ Write as common base, } \frac{1}{27}=3^{-3} \\ 3^x=3^{-3} \amp\quad \text{ Same base, set exponents equal } \\ x=-3 \amp\quad \text{ Our Solution } \end{align*}
    World View Note: Dutch mathematician Adriaan Vlacq published a text in \(1628\) which listed logarithms calculated out from \(1\) to \(100,000\text{!}\)
    Solving equations with logarithms is done in a very similar way, we simply will change the equation into exponential form and try to solve the resulting equation.

    Example 10.5.6.

    \begin{align*} \log_5 x=2 \amp \quad \text{ Change to exponential form } \\ 5^2=x \amp\quad \text{ Evaluate exponent } \\ 25=x \amp\quad \text{ Our solution } \end{align*}

    Example 10.5.7.

    \begin{align*} \log_2 (3x+5)=4 \amp \quad \text{ Change to exponential form } \\ 2^4=3x+5 \amp\quad \text{ Evaluate exponent } \\ 16=3x+5 \amp\quad \text{ Solve } \\ \underline{-5 \quad\quad -5} \amp\quad \text{ Subtract $5$ from both sides } \\ 11=3x \amp\quad \text{ Divide both sides by } 3 \\ \overline{\ 3\ } \quad \overline{\ 3\ } \amp\quad \text{ } \\ \frac{11}{3}=x \amp\quad \text{ Our Solution } \end{align*}

    Example 10.5.8.

    \begin{align*} \log_x 8=3 \amp \quad \text{ Change to exponential form } \\ x^3=8 \amp\quad \text{ Cube root of both sides } \\ x=2 \amp\quad \text{ Our solution } \end{align*}
    There is one base on a logarithm that gets used more often than any other base, base \(10\text{.}\) Similar to square roots not writing the common index of \(2\) in the radical, we don’t write the common base of \(10\) in the logarithm. So if we are working on a problem with no base written we will always assume that base is base \(10\text{.}\)

    Example 10.5.9.

    \begin{align*} \log x=-2 \amp \quad \text{ Rewrite as exponent, $10$ is base } \\ 10^{-2} =x \amp\quad \text{ Evaluate, remember negative exponent is fraction } \\ \frac{1}{100}=x \amp\quad \text{ Our solution } \end{align*}
    This lesson has introduced the idea of logarithms, changing between logs and exponents, evaluating logarithms, and solving basic logarithmic equations. In an advanced algebra course, logarithms will be studied in much greater detail.
