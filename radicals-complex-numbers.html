    Section 8.8 Complex Numbers

    Objective: Add, subtract, multiply, rationalize, and simplify expres- sions using complex numbers.
    World View Note: When mathematics was first used, the primary purpose was for counting. Thus they did not originally use negatives, zero, fractions or irrational numbers. However, the ancient Egyptians quickly developed the need for "a part" and so they made up a new type of number, the ratio or fraction. The Ancient Greeks did not believe in irrational numbers (people were killed for believing otherwise). The Mayans of Central America later made up the number zero when they found use for it as a placeholder. Ancient Chinese Mathematicians made up negative numbers when they found use for them.
    In mathematics, when the current number system does not provide the tools to solve the problems the culture is working with, we tend to make up new ways for dealing with the problem that can solve the problem. Throughout history this has been the case with the need for a number that is nothing (\(0\)), smaller than zero (negatives), between integers (fractions), and between fractions (irrational numbers). This is also the case for the square roots of negative numbers. To work with the square root of negative numbers mathematicians have defined what are called imaginary and complex numbers
    \begin{equation*} \textbf{ Definition of Imaginary Numbers: } \mathbf{i^2=-1} \textbf{ (thus } \mathbf{i=\sqrt{-1}}\textbf{)} \end{equation*}
    Examples of imaginary numbers include \(3i\text{,}\) \(- 6i\text{,}\) \(\frac{3}{5}i\) and \(3i\sqrt{5}\text{.}\) A complex number is one that contains both a real and imaginary part, such as \(2 + 5i\text{.}\)
    With this definition, the square root of a negative number is no longer undefined. We now are allowed to do basic operations with the square root of negatives. First we will consider exponents on imaginary numbers. We will do this by manipulating our definition of \(i^2 = - 1\text{.}\) If we multiply both sides of the definition by \(i\text{,}\) the equation becomes \(i^3 = - i\text{.}\) Then if we multiply both sides of the equation again by \(i\text{,}\) the equation becomes \(i^4 =-i^2 =-(-1)=1\text{,}\) or simply \(i^4 =1\text{.}\) Multiplying again by \(i\) gives \(i^5 = i\text{.}\) One more time gives \(i^6 = i^2 = - 1\text{.}\) And if this pattern continues we see a cycle forming, the exponents on i change we cycle through simplified answers of \(i\text{,}\) \(- 1\text{,}\) \(- i\text{,}\) \(1\text{.}\) As there are \(4\) different possible answers in this cycle, if we divide the exponent by \(4\) and consider the remainder, we can simplify any exponent on \(i\) by learning just the following four values:

    Example 8.8.2.

    \begin{align*} i^{35} \amp \quad \text{ Divide exponent by } 4 \\ 8R3 \amp\quad \text{ Use remainder as exponent on } i \\ i^3 \amp\quad \text{ Simplify } \\ -i \amp\quad \text{ Our Solution } \end{align*}

    Example 8.8.3.

    \begin{align*} i^{124} \amp \quad \text{ Divide exponent by } 4 \\ 31R0 \amp\quad \text{ Use remainder as exponent on } i \\ i^0 \amp\quad \text{ Simplify } \\ 1 \amp\quad \text{ Our Solution } \end{align*}
    When performing operations (add, subtract, multilpy, divide) we can handle i just like we handle any other variable. This means when adding and subtracting complex numbers we simply add or combine like terms.

    Example 8.8.4.

    \begin{align*} (2+5i)+(4-7i) \amp \quad \text{ Combine like terms $2+4$ and $5i-7i$ } \\ 6 - 2i \amp\quad \text{ Our Solution } \end{align*}
    It is important to notice what operation we are doing. Students often see the parenthesis and think that means FOIL. We only use FOIL to multiply. This problem is an addition problem so we simply add the terms, or combine like terms.
    For subtraction problems the idea is the same, we need to remember to first distribute the negative onto all the terms in the parentheses.

    Example 8.8.5.

    \begin{align*} (4 - 8i) - (3 - 5i)\amp \quad \text{ Distribute the negative } \\ 4-8i-3+5i\amp\quad \text{ Combine like terms $4-3$ and $-8i+5i$ } \\ 1 - 3i \amp\quad \text{ Our Solution } \end{align*}
    Addition and subtraction can be combined into one problem.

    Example 8.8.6.

    \begin{align*} (5i) - (3 + 8i) + ( - 4 + 7i)\amp \quad \text{ Distribute the negative } \\ 5i-3-8i-4+7i\amp\quad \text{ Combine like terms $5i-8i+7i$ and $-3-4$ } \\ - 7 + 4i \amp\quad \text{ Our Solution } \end{align*}
    Multiplying with complex numbers is the same as multiplying with variables with one exception, we will want to simplify our final answer so there are no exponents on \(i\text{.}\)

    Example 8.8.7.

    \begin{align*} (3i)(7i) \amp \quad \text{ Multilpy coefficients and $i$'s } \\ 21i^2 \amp\quad \text{ Simplify } i^2 = - 1 \\ 21( - 1) \amp\quad \text{ Multiply } \\ - 21 \amp\quad \text{ Our Solution } \end{align*}

    Example 8.8.8.

    \begin{align*} 5i(3i - 7) \amp \quad \text{ Distribute } \\ 15i^2 - 35i \amp\quad \text{ Simplify } i^2 = - 1 \\ 15(-1)-35i \amp\quad \text{ Multiply } \\ -15-35i \amp\quad \text{ Our Solution } \end{align*}

    Example 8.8.9.

    \begin{align*} (2-4i)(3+5i) \amp \quad \text{ FOIL } \\ 6+10i-12i-20i^2\amp\quad \text{ Simplify } i^2 = - 1 \\ 6+10i-12i-20(-1) \amp\quad \text{ Multiply } \\ 6+10i-12i+20\amp\quad \text{ Combine like terms $6+20$ and $10i-12i$} \\ 26-2i \amp\quad \text{ Our Solution } \end{align*}

    Example 8.8.10.

    \begin{align*} (2-4i)(3+5i)(3i)(6i)(2 - 3i) \amp \quad \text{ Multiply first two monomials } \\ 18i^2(2 - 3i)\amp\quad \text{ Distribute } \\ 36i^2 - 54i^3 \amp\quad \text{ Simplify $i^2=-1$ and $i^3=-i$ } \\ 36(-1)-54(-i) \amp\quad \text{ Multiply} \\ -36+54i \amp\quad \text{ Our Solution } \end{align*}
    Remember when squaring a binomial we either have to FOIL or use our shortcut to square the first, twice the product and square the last. The next example uses the shortcut.

    Example 8.8.11.

    \begin{align*} (4 - 5i)^2 \amp \quad \text{ Use perfect square shortcut } \\ 4^2=16 \amp\quad \text{ Square the first } \\ 2(4)(-5i)=-40i \amp\quad \text{ Twice the product } \\ (5i)^2 =25i^2 =25(-1)=-25 \amp\quad \text{ Square the last, simplify } i^2 = - 1 \\ 16-40i-25 \amp\quad \text{ Combine like terms} \\ -9-40i \amp\quad \text{ Our Solution } \end{align*}
    Dividing with complex numbers also has one thing we need to be careful of. If \(i\) is \(\sqrt{-1}\text{,}\) and it is in the denominator of a fraction, then we have a radical in the denominator! This means we will want to rationalize our denominator so there are no \(i\)’s. This is done the same way we rationalized denominators with square roots.

    Example 8.8.12.

    \begin{align*} \frac{7+3i}{-5i} \amp \quad \text{ Just a monomial in denominator, multiply by } i \\ \frac{7+3i}{-5i} \left(\frac{i}{i}\right) \amp\quad \text{ Distribute $i$ in numerator } \\ \frac{7i+3i^2}{-5i^2} \amp\quad \text{ Simplify } i^2=-1 \\ \frac{7i+3(-1)}{-5(-1)}\amp\quad \text{ Multiply } \\ \frac{7i-3}{5} \amp\quad \text{ Our Solution } \end{align*}
    The solution for these problems can be written several different ways, for example \(\frac{-3+7i}{5}\) or \(\frac{-3}{5} + \frac{7}{5}i\text{.}\) The author has elected to use the solution as written, but it is important to express your answer in the form your instructor prefers.

    Example 8.8.13.

    \begin{align*} \frac{2-6i}{4+8i} \amp \quad \text{ Binomial in denominator, multiply by conjugate, } 4 - 8i \\ \frac{2-6i}{4+8i}\left(\frac{4-8i}{4-8i}\right) \amp\quad \text{ FOIL in numerator, denominator is a difference of squares } \\ \frac{8-16i-24i+48i^2}{16 - 64i^2} \amp\quad \text{ Simplify } i^2=-1\\ \frac{8-16i-24i+48(-1)}{16-64(-1)} \amp\quad \text{ Multiply } \\ \frac{8-16i-24i-48}{16+64} \amp\quad \text{ Combine like terms $8-48$ and $-16i-24i$ and }16+64 \\ \frac{-40-40i}{80} \amp\quad \text{ Reduce, divide each term by } 40 \\ \frac{-1-i}{2} \amp\quad \text{ Our Solution } \end{align*}
    Using \(i\) we can simplify radicals with negatives under the root. We will use the product rule and simplify the negative as a factor of negative one. This is shown in the following examples.

    Example 8.8.14.

    \begin{align*} \sqrt{-16} \amp \quad \text{ Consider the negative as a factor of } -1 \\ \sqrt{-1\cdot 16} \amp\quad \text{ Take each root, square root of $-1$ is } i \\ 4i \amp\quad \text{ Our Solution } \end{align*}

    Example 8.8.15.

    \begin{align*} \sqrt{-24} \amp \quad \text{ Find perfect square factors, including } -1 \\ \sqrt{-1\cdot 4\cdot 6} \amp\quad \text{ Square root of $-1$ is $i$, square root of $4$ is $2$ } \\ 2i\sqrt{6} \amp\quad \text{ Our Solution } \end{align*}
    When simplifying complex radicals it is important that we take the \(- 1\) out of the radical (as an \(i\)) before we combine radicals.

    Example 8.8.16.

    \begin{align*} \sqrt{-6}\sqrt{-3} \amp \quad \text{ Simplify the negatives, bringing $i$ out of radicals } \\ \left(i\sqrt{6}\right)\left(i\sqrt{3}\right) \amp\quad \text{ Multiply $i$ by $i$ is $i^2 = - 1$, also multiply radicals } \\ -\sqrt{18} \amp\quad \text{ Simplify the radical } \\ -\sqrt{9\cdot 2} \amp\quad \text{ Take square root of } 9 \\ -3\sqrt{ 2} \amp\quad \text{ Our Solution } \end{align*}
    If there are fractions, we need to make sure to reduce each term by the same number. This is shown in the following example.

    Example 8.8.17.

    \begin{align*} \frac{-15-\sqrt{-200}}{20} \amp \quad \text{ Simplify the radical first } \\ \sqrt{-200} \amp\quad \text{ Find perfect square factors, including } -1 \\ \sqrt{-1\cdot 100\cdot 2} \amp\quad \text{ Take square root of $-1$ and $100$ } \\ 10i\sqrt{2} \amp\quad \text{ Put this back into the expression } \\ \frac{-15-10i\sqrt{2}}{20} \amp\quad \text{All the factors are divisible by } 5 \\ \frac{-3-2i\sqrt{2}}{4} \amp\quad \text{ Our Solution } \end{align*}
    By using \(i = \sqrt{- 1}\) we will be able to simplify and solve problems that we could not simplify and solve before. This will be explored in more detail in a later section.
