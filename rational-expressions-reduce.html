    Section 7.1 Reduce Rational Expressions

    Objective: Reduce rational expressions by dividing out common factors.
    Rational expressions are expressions written as a quotient of polynomials. Examples of rational expressions include:
    \begin{gather*} \frac{x^2-x-12}{x^2-9x+20} \quad \text{ and } \quad \frac{3}{x-2} \quad \text{ and } \quad \frac{a-b}{b-a} \quad \text{ and } \quad \frac{3}{2} \end{gather*}
    As rational expressions are a special type of fraction, it is important to remember with fractions we cannot have zero in the denominator of a fraction. For this reason, rational expressions may have one more excluded values, or values that the variable cannot be or the expression would be undefined.

    Example 7.1.1.

    State the excluded value(s):
    \begin{align*} \frac{x^2-1}{3x^2+5x} \amp \quad \text{ Denominator cannot be zero } \\ 3x^2+5x\neq 0 \amp\quad \text{ Factor } \\ x\left(3x+5\right)\neq 0 \amp\quad \text{ Set each factor not equal to zero } \\ x\neq 0 \text{ or } 3x+5\neq 0 \amp\quad \text{ Subtract $5$ from second equation } \\ -5\ -5 \amp\quad \text{ } \\ x\neq 0 \text{ or } 3x\neq -5 \amp\quad \text{ Divide second equation by } 3 \\ x\neq 0 \text{ or } x\neq \frac{-5}{3} \amp\quad \text{ Our Solution } \end{align*}
    This means we can use any value for \(x\) in the equation except for \(0\) and \(\frac{-5}{3}\text{.}\) We can however, evaluate any other value in the expression.
    World View Note: The number zero was not widely accepted in mathematical thought around the world for many years. It was the Mayans of Central America who first used zero to aid in the use of their base \(20\) system as a place holder!
    Rational expressions are easily evaluated by simply substituting the value for the variable and using order of operations.

    Example 7.1.2.

    Determine the value of the following expression for \(x=-6\text{:}\)
    \begin{align*} \frac{x^2-4}{x^2+6x+8} \amp \quad \text{ Substitute $-6$ for each occurence of variable $x$ } \\ \frac{(-6)^2-4}{(-6)^2+6(-6)+8} \amp\quad \text{ Exponents first } \\ \frac{36-4}{36+6(-6)+8} \amp\quad \text{ Multiply } \\ \frac{36-4}{36-36+8} \amp\quad \text{ Add and subtract } \\ \frac{32}{8} \amp\quad \text{ Reduce } \\ 4 \amp\quad \text{ Our Solution } \end{align*}
    Just as we reduced the previous example, often a rational expression can be reduced, even without knowing the value of the variable. When we reduce we divide out common factors. We have already seen this with monomials when we discussed properties of exponents. If the problem only has monomials we can reduce the coefficients, and subtract exponents on the variables.

    Example 7.1.3.

    \begin{align*} \frac{15x^4y^2}{25x^2y^6} \amp \quad \text{ Reduce, subtract exponents. Negative exponents move to denominator } \\ \frac{3x^2}{5y^4} \amp\quad \text{ Our Solution } \end{align*}
    However, if there is more than just one term in either the numerator or denominator, we can’t divide out common factors unless we first factor the numerator and denominator.

    Example 7.1.4.

    \begin{align*} \frac{28}{8x^2-16} \amp \quad \text{ Denominator has a common factor of } 8 \\ \frac{28}{8(x^2-2)} \amp\quad \text{ Reduce by dividing $28$ and $8$ by } 4 \\ \frac{7}{2(x^2-2)} \amp\quad \text{ Our Solution } \end{align*}

    Example 7.1.5.

    \begin{align*} \frac{9x-3}{18x-6} \amp \quad \text{ Numerator has a common factor of $3$, denominator of } 6 \\ \frac{3(3x-1)}{6(3x-1)} \amp\quad \text{ Divide out common factor $(3x - 1)$ and divide $3$ and $6$ by } 3 \\ \frac{1}{2} \amp\quad \text{ Our Solution } \end{align*}

    Example 7.1.6.

    \begin{align*} \frac{x^2 - 25}{x^2 +8x+15} \amp \quad \text{ Numerator is difference of squares, denominator is factored using AC method} \\ \frac{(x+5)(x-5)}{(x+3)(x+5)} \amp\quad \text{ Divide out common factor } (x + 5) \\ \frac{x-5}{x+3} \amp\quad \text{ Our Solution } \end{align*}
    It is important to remember we cannot reduce terms, only factors. This means if there are any \(+\) or \(-\) between the parts we want to reduce we cannot. In the previous example we had the solution \(x-5\text{,}\) we cannot divide out the \(x\)’s because \(x+3\) they are terms (separated by \(+\) or \(-\) ) not factors (separated by multiplication).
