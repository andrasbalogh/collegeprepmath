    Section 10.3 Inverse Functions

    Objective: Identify and find inverse functions.
    When a value goes into a function it is called the input. The result that we get when we evaluate the function is called the output. When working with functions sometimes we will know the output and be interested in what input gave us the output. To find this we use an inverse function. As the name suggests an inverse function undoes whatever the function did. If a function is named \(f(x)\text{,}\) the inverse function will be named \(f^{-1}(x)\) (read “f inverse of x”). The negative one is not an exponent, but merely a symbol to let us know that this function is the inverse of \(f\text{.}\)
    World View Note: The notation used for functions was first introduced by the great Swiss mathematician, Leonhard Euler in the \(18^{\text{th}}\) century.
    For example, if \(f(x)=x+5\text{,}\) we could deduce that the inverse function would be \(f^{-1}(x)=x-5\text{.}\) If we had an input of \(3\text{,}\) we could calculate \(f(3)=(3)+5=8\text{.}\) Our output is 8. If we plug this output into the inverse function we get \(f ^{-1}(8)=(8)-5=3\text{,}\) which is the original input.
    Often the functions are much more involved than those described above. It may be difficult to determine just by looking at the functions if they are inverses. In order to test if two functions, \(f(x)\) and \(g(x)\) are inverses we will calculate the composition of the two functions at \(x\text{.}\) If \(f\) changes the variable \(x\) in some way, then \(g\) undoes whatever \(f\) did, then we will be back at \(x\) again for our final solution. In other words, if we simplify \((f \circ g)(x)\) the solution will be \(x\text{.}\) If it is anything but \(x\) the functions are not inverses.

    Example 10.3.1.

    \begin{align*} \text{Are } f(x)=\sqrt[3]{3x+4} \text{ and } g(x)= \frac{x^3-4}{3} \text{ inverses?} \amp \quad \text{ Calculate composition } \\ f\left(g(x)\right) \amp\quad \text{ Replace $g(x)$ with } \frac{x^3-4}{3}\\ f\left(\frac{x^3-4}{3}\right) \amp\quad \text{ Substitute} \frac{x^3-4}{3} \text{for variable in }f \\ \sqrt[3]{3\left(\frac{x^3-4}{3}\right)+4} \amp\quad \text{ Divide out the $3$’s } \\ \sqrt[3]{x^3-4+4} \amp\quad \text{ Combine like terms } \\ \sqrt[3]{x^3} \amp\quad \text{ Take cube root } \\ x \amp\quad \text{ Simplified to $x$! } \\ \text{Yes, they are inverses!} \amp\quad \text{ Our Solution } \end{align*}

    Example 10.3.2.

    \begin{align*} \text{Are } h(x)=2x+5 \text{ and } g(x)= \frac{x}{2}-5 \text{ inverses?} \amp \quad \text{Calculate composition } \\ h\left(g(x)\right) \amp\quad \text{Replace $g(x)$ with } \frac{x}{2}-5 \\ h\left(\frac{x}{2}-5\right) \amp\quad \text{Substitute } \frac{x}{2}-5 \text{ for variable in } h \\ 2\left(\frac{x}{2}-5\right)+5 \amp\quad \text{Distrubte $2$ } \\ x-10+5 \amp\quad \text{Combine like terms } \\ x-5 \amp\quad \text{Did not simplify to } x \\ \text{No, they are not inverses} \amp\quad \text{ Our Solution } \end{align*}

    Example 10.3.3.

    \begin{align*} \text{Are } f(x)= \frac{3x-2}{4x+1} \text{ and } g(x)= \frac{x+2}{3-4x} \text{ inverses?} \amp \quad \text{Calculate composition } \\ f\left(g(x)\right) \amp\quad \text{Replace $g(x)$ with } \frac{x+2}{3-4x} \\ f\left(\frac{x+2}{3-4x}\right) \amp\quad \text{Substitute} \frac{x+2}{3-4x} \text{for variable in } f \\ \\ \frac{3\left(\frac{x+2}{3-4x}\right)-2}{4\left(\frac{x+2}{3-4x}\right)+1} \amp\quad \text{Distrubte $3$ and $4$ into numerators } \\ \\ \frac{\left(\frac{3x+6}{3-4x}\right)-2}{\left(\frac{4x+8}{3-4x}\right)+1} \amp\quad \text{Multiply each term by LCD: }3-4x \\ \\ \frac{\frac{(3x+6)(3-4x)}{3-4x}-2(3-4x)}{\frac{(4x+8)(3-4x)}{3-4x}+1(3-4x)} \amp\quad \text{Reduce fractions } \\ \\ \frac{3x+6-2(3-4x)}{4x+8+1(3-4x)} \amp\quad \text{Distribute } \\ \\ \frac{3x+6-6+8x}{4x+8+3-4x} \amp\quad \text{Combine like terms } \\ \frac{11x}{11} \amp\quad \text{Divide out } 11 \\ \\ x \amp\quad \text{Simplified to $x$! } \\ \text{Yes, they are inverses!} \amp\quad \text{Our Solution } \end{align*}
    While the composition is useful to show two functions are inverses, a more common problem is to find the inverse of a function. If we think of \(x\) as our input and \(y\) as our output from a function, then the inverse will take \(y\) as an input and give \(x\) as the output. This means if we switch \(x\) and \(y\) in our function we will find the inverse! This process is called the switch and solve strategy.
    Switch and solve strategy to find an inverse:
    1. Replace \(f(x)\) with \(y\)
    2. Switch \(x \) and \(y\)’s
    3. Solve for \(y \)
    4. Replace \(y\) with \(f^{-1}(x) \)

    Example 10.3.4.

    \begin{align*} \text{Find the inverse of } f(x)=(x+4)^3-2 \amp \quad \text{Replace $f(x)$ with $y$ } \\ y=(x+4^3)-2 \amp\quad \text{Switch $x$ and $y$ } \\ x=(y+4)^3-2 \amp\quad \text{Solve for} y \\ \underline{+2 \qquad \qquad +2} \amp\quad \text{Add $2$ to both sides } \\ x+2=(y+4)^3 \amp\quad \text{Cube root both sides } \\ \sqrt[3]{x+2}=y+4 \amp\quad \text{Subtract $4$ from both sides } \\ \underline{-4 \qquad -4} \\ \sqrt[3]{x-2}-4=y \amp\quad \text{Replacing $y$ with } f^{-1}(x) \\ f^{-1}(x)=\sqrt[3]{x+2}-4 \amp\quad \text{Our Solution } \end{align*}

    Example 10.3.5.

    \begin{align*} \text{Find the inverse of } g(x)=\frac{2x-3}{4x+2} \amp \quad \text{Replace $g(x)$ with $y$ } \\ \\ y=\frac{2x-3}{4x+2} \amp\quad \text{Switch $x$ and $y$ } \\ \\ x=\frac{2y-3}{4y+2} \amp\quad \text{Multiply } (4y+2) \\ \\ x(4y+2)=2y-3 \amp\quad \text{Distrubute } \\ 4xy+2x=2y-3 \amp\quad \text{ Move all $y$’s to one side, rest to other side } \\ \underline{-4xy+3\ -4xy+3} \amp\quad \text{ Subtract $4xy$ and add $3$ to both sides } \\ 2x+3=2y-4xy \amp\quad \text{ Factor out }y \\ \\ 2x+3=y(2-4x) \amp\quad \text{ Divide by }2-4x \\ \overline{2-4x}\qquad \quad \overline{2-4x}\\ \\ \frac{2x+3}{2-4x}=y \amp\quad \text{ Replacing $y$ with } g^{-1}(x) \\ \\ g^{-1}(x)=\frac{2x+3}{2-4x} \amp\quad \text{ Our Solution } \end{align*}
    In this lesson we looked at two different things, first showing functions are inverses by calculating the composition, and second finding an inverse when we only have one function. Be careful not to get them backwards. When we already have two functions and are asked to show they are inverses, we do not want to use the switch and solve strategy, what we want to do is calculate the inverse. There may be several ways to represent the same function so the switch and solve strategy may not look the way we expect and can lead us to conclude two functions are not inverses when they are in fact inverses.
