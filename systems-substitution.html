    Section 4.2 Substitution

    Objective: Solve systems of equations using substitution.
    When solving a system by graphing has several limitations. First, it requires the graph to be perfectly drawn, if the lines are not straight we may arrive at the wrong answer. Second, graphing is not a great method to use if the answer is really large, over \(100\) for example, or if the answer is a decimal the that graph will not help us find, \(3.2134\) for example. For these reasons we will rarely use graphing to solve our systems. Instead, an algebraic approach will be used.
    The first algebraic approach is called substitution. We will build the concepts of substitution through several example, then end with a five-step process to solve problems using this method.

    Example 4.2.1.

    \begin{align*} \begin{matrix} x=5 \\ y=2x-3 \end{matrix} \amp \quad \text{ We already know $x = 5$, substitute this into the other equation } \\ \\ y=2(5)-3 \amp\quad \text{ Evaluate, multiply first } \\ y=10-3 \amp\quad \text{ Subtract } \\ y=7 \amp\quad \text{ We now also have } y \\ (5, 7) \amp\quad \text{ Our Solution } \end{align*}
    When we know what one variable equals we can plug that value (or expression) in for the variable in the other equation. It is very important that when we substitute, the substituted value goes in parenthesis. The reason for this is shown in the next example.

    Example 4.2.2.

    \begin{align*} \begin{matrix} 2x-3y=7 \\ y=3x-7 \end{matrix} \amp \quad \text{ We know $y = 3x - 7$, substitute this into the other equation } \\ 2x - 3(3x - 7) = 7 \amp\quad \text{ Solve this equation, distributing $- 3$ first } \\ 2x-9x+21=7 \amp\quad \text{ Combine like terms } 2x-9x\\ -7x+21=7\amp\quad \text{ Subtract } 21 \\ -21\ -21 \amp\quad \text{ } \\ \frac{-7x}{-7}=\frac{-14}{-7} \amp\quad \text{ Divideby } -7 \\ x=2 \amp\quad \text{ We now have our $x$, plugin to the $y$-equation to find } y \\ y=3(2)-7 \amp\quad \text{ Evaluate, multiply first } \\ y=6-7 \amp\quad \text{ Subtract } \\ y=-1 \amp\quad \text{ We now also have } y \\ (2,-1) \amp\quad \text{ Our Solution } \end{align*}
    By using the entire expression \(3x - 7\) to replace \(y\) in the other equation we were able to reduce the system to a single linear equation which we can easily solve for our first variable. However, the lone variable (a variable without a coefficient) is not always alone on one side of the equation. If this happens we can isolate it by solving for the lone variable.

    Example 4.2.3.

    \begin{align*} \begin{matrix} 3x+2y=1 \\ {\mathbf x} -5y=6 \end{matrix} \amp \quad \text{ Lone variable is $x$, isolate by adding $5y$ to both sides. } \\ +5y\ +5y \amp\quad \text{ } \\ x=6+5y \amp\quad \text{ Substitute this into the untouched equation } \\ 3(6+5y)+2y=1 \amp\quad \text{ Solve this equation, distributing $3$ first } \\ 18+15y+2y=1 \amp\quad \text{ Combine like terms }15y+2y \\ 18+17y=1 \amp\quad \text{ Subtract $18$ from both sides } \\ -18 \qquad -18 \amp\quad \text{ } \\ \frac{17y}{17}=\frac{-17}{17} \amp\quad \text{ Divide both sides by } 17 \\ y=-1 \amp\quad \text{ We have our $y$, plug this into the $x$-equation to find } x \\ x=6+5(-1) \amp\quad \text{ Evaluate, multiply first } \\ x=6-5 \amp\quad \text{ Subtract } \\ x=1 \amp\quad \text{ We now also have } x \\ (1,-1) \amp\quad \text{ Our Solution } \end{align*}
    The process in the previous example is how we will solve problems using substitution. This process is described and illustrated in the following table which lists the five steps to solving by substitution.
    Sometimes we have several lone variables in a problem. In this case we will have the choice on which lone variable we wish to solve for, either will give the same final result.

    Example 4.2.5.

    \begin{align*} x+y=5 \amp \quad \text{ Find the lone variable: $x$ or $y$ in first, or $x$ in second. } \\ x-y=-1 \amp\quad \text{ We will chose $x$ in the first } \\ \\ x+y=5 \amp\quad \text{ Solve for the lone variable, subtract $y$ from both sides } \\ -y\ -y \amp\quad \text{ } \\ x=5-y \amp\quad \text{ Plug into the untouched equation, the second equation } \\ (5-y)-y=-1 \amp\quad \text{ Solve, parenthesis are not needed here, combine like terms } \\ 5-2y=-1 \amp\quad \text{ Subtract $5$ from both sides } \\ -5\qquad \quad -5 \amp\quad \text{ } \\ \frac{-2y}{-2}=\frac{-6}{-2} \amp\quad \text{Divide both sides by } -2 \\ y=3 \amp\quad \text{ We have our } y \\ x=5-(3) \amp\quad \text{Plug into lone variable equation, evaluate} \\ x=2 \amp\quad \text{ Now we have our } x \\ (2, 3) \amp\quad \text{ Our Solution } \end{align*}
    Just as with graphing it is possible to have no solution \(\emptyset\) (parallel lines) or infinite solutions (same line) with the substitution method. While we won’t have a parallel line or the same line to look at and conclude if it is one or the other, the process takes an interesting turn as shown in the following example.

    Example 4.2.6.

    \begin{align*} \begin{matrix} {\mathbf y}+4=3x\\ 2y - 6x = - 8 \end{matrix} \amp \quad \text{ Find the lone variable, $y$ in the first equation } \\ y+4=3x \amp\quad \text{ Solve for the lone variable, subtract $4$ from both sides } \\ -4\quad -4 \amp\quad \text{ } \\ y=3x-4 \amp\quad \text{ Plug into untouched equation } \\ 2(3x-4)-6x=-8 \amp\quad \text{ Solve, distribute through parenthesis } \\ 6x-8-6x=-8 \amp\quad \text{ Combine like terms 6x-6x } \\ -8=-8 \amp\quad \text{Variables are gone! A true statement.} \\ \text{Infinite solutions} \amp\quad \text{ Our Solution } \end{align*}
    Because we had a true statement, and no variables, we know that anything that works in the first equation, will also work in the second equation. However, we do not always end up with a true statement.

    Example 4.2.7.

    \begin{align*} \begin{matrix} 6x-3y=-9 \\ -2x+y=5 \end{matrix} \amp \quad \text{ Find the lone variable, $y$ in the second equation } \\ -2x+{\mathbf y}=5 \amp\quad \text{ Solve for the lone variable, add $2x$ to both sides } \\ +2x\qquad +2x \amp\quad \text{ } \\ y=5+2x \amp\quad \text{ Plug into untouched equation } \\ 6x-3(5+2x)=-9 \amp\quad \text{ Solve, distribute through parenthesis } \\ 6x-15-6x=-9 \amp\quad \text{ Combine like terms } 6x-6x \\ -15\neq -9 \amp\quad \text{ Variables are gone! A false statement. } \\ \text{No Solution } \emptyset \amp\quad \text{Our Solution} \end{align*}
    Because we had a false statement, and no variables, we know that nothing will work in both equations.
    World View Note: French mathematician Rene Descartes wrote a book which included an appendix on geometry. It was in this book that he suggested using letters from the end of the alphabet for unknown values. This is why often we are solving for the variables \(x\text{,}\) \(y\text{,}\) and \(z\text{.}\)
    One more question needs to be considered, what if there is no lone variable? If there is no lone variable substitution can still work to solve, we will just have to select one variable to solve for and use fractions as we solve.

    Example 4.2.8.

    \begin{align*} 5x - 6y = - 14 \amp \quad \text{No lone variable,} \\ -2x+4y=12 \amp\quad \text{ we will solve for $x$ in the first equation.} \\ \amp\quad \text{ } \\ 5x-6y=-14 \amp\quad \text{ Solve for our variable, add $6y$ to both sides } \\ +6y\quad +6y \amp\quad \text{ } \\ \frac{5x}{5}=\frac{-14}{5}+\frac{6y}{5} \amp\quad \text{ Divide each term by } 5 \\ x=\frac{-14}{5}+\frac{6y}{5} \amp\quad \text{ Plug into untouched equation } \\ -2\left(\frac{-14}{5}+\frac{6y}{5}\right)+4y=12 \amp\quad \text{ Solve, distribute through parenthesis } \\ \frac{28}{5}- \frac{12y}{5}+4y=12 \amp\quad \text{ Clear fractions by multiplying by } 5 \\ \frac{28(5)}{5}- \frac{12y(5)}{5}+4y(5)=12(5) \amp\quad \text{ Reduce fractions and multiply } \\ 28-12y+20y=60 \amp\quad \text{ Combine like terms } - 12y + 20y \\ 28+8y=60 \amp\quad \text{ Subtract 28 from both sides } \\ -28 \qquad -28 \amp\quad \text{ } \\ \frac{8y}{8}=\frac{32}{8} \amp\quad \text{Divide both sides by } 8 \\ y=4 \amp\quad \text{We have our } y \\ x=\frac{-14}{5}+\frac{6(4)}{5} \amp\quad \text{ Plug into lone variable equation, multiply } \\ x=\frac{-14}{5}+\frac{24}{5} \amp\quad \text{ Add fractions } \\ x=\frac{10}{5} \amp\quad \text{ Reduce fraction } \\ x=2 \amp\quad \text{ Now we have our } x \\ (x,y)=(2,4) \amp\quad \text{ Our solution } \end{align*}
    Using the fractions does make the problem a bit more tricky. This is why we have another method for solving systems of equations that will be discussed in another lesson.
