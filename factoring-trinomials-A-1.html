    Section 6.3 Trinomials where \(a= 1\)

    Objective: Factor trinomials where the coefficient of \(x^2\) is one.
    Factoring with three terms, or trinomials, is the most important type of factoring to be able to master. As factoring is multiplication backwards we will start with a multipication problem and look at how we can reverse the process.

    Example 6.3.1.

    \begin{align*} (x + 6)(x - 4) \amp \quad \text{ Distribute $(x + 6)$ through second parenthesis } \\ x(x + 6) - 4(x + 6) \amp\quad \text{ Distribute each monomial through parenthesis } \\ x^2 + 6x - 4x - 24 \amp\quad \text{ Combine like terms } \\ x^2 + 2x - 24\amp\quad \text{ Our Solution } \end{align*}
    You may notice that if you reverse the last three steps the process looks like grouping. This is because it is grouping! The GCF of the left two terms is \(x\) and the GCF of the second two terms is \(- 4\text{.}\) The way we will factor trinomials is to make them into a polynomial with four terms and then factor by grouping. This is shown in the following example, the same problem worked backwards.

    Example 6.3.2.

    \begin{align*} x^2+2x-24 \amp \quad \text{ Split middle term into } + 6x - 4x \\ x^2+6x-4x-24 \amp\quad \text{ Grouping: GCF on left is x, on right is } -4 \\ x(x+6)-4(x+6) \amp\quad (x+6) \text{ is the same, factor out this GCF } \\ (x+6)(x-4) \amp\quad \text{ Our Solution } \end{align*}
    The trick to make these problems work is how we split the middle term. Why did we pick \(+6x-4x\) and not \(+5x-3x\text{?}\) The reason is because \(6x-4x\) is the only combination that works! So how do we know what is the one combination that works? To find the correct way to split the middle term we will use what is called the ac method. In the next lesson we will discuss why it is called the ac method. The way the ac method works is we find a pair of numers that multiply to a cer- tain number and add to another number. Here we will try to multiply to get the last term and add to get the coefficient of the middle term. In the previous example that would mean we wanted to multiply to \(- 24\) and add to \(+ 2\text{.}\) The only numbers that can do this are \(6\) and \(-4\text{:}\) \((6\times(-4)=-24\) and \(6+(-4)=2)\text{.}\) This process is shown in the next few examples.

    Example 6.3.3.

    \begin{align*} x^2 + 9x + 18\amp \quad \text{ Want to multiply to $18$, add to $9$ } \\ x^2 +6x+3x+18 \amp\quad \text{ $6$ and $3$, split the middle term } \\ x(x + 6) + 3(x + 6) \amp\quad \text{ Factor by grouping } \\ (x + 6)(x + 3) \amp\quad \text{ Our Solution } \end{align*}

    Example 6.3.4.

    \begin{align*} x^2 -4x+3 \amp \quad \text{ Want to multiply to $3$, add to } - 4 \\ x2-3x-x+3 \amp\quad \text{ $-3$ and $-1$, split the middle term } \\ x(x-3)-1(x-3) \amp\quad \text{ Factor by grouping } \\ (x-3)(x-1) \amp\quad \text{ Our Solution } \end{align*}

    Example 6.3.5.

    \begin{align*} x^2 - 8x - 20 \amp \quad \text{ Want to multiply to $- 20$, add to } - 8 \\ x^2-10x+2x-20 \amp\quad \text{ $-10$ and $2$, split the middle term } \\ x(x - 10) + 2(x - 10) \amp\quad \text{ Factor by grouping } \\ (x - 10)(x + 2) \amp\quad \text{ Our Solution } \end{align*}
    Often when factoring we have two variables. These problems solve just like prob- lems with one variable, using the coefficients to decide how to split the middle term.

    Example 6.3.6.

    \begin{align*} a^2 -9ab+14b^2 \amp \quad \text{ Want to multiply to $14$, add to } -9 \\ a^2 - 7a b - 2a b + 14b^2 \amp\quad \text{ $- 7$ and $- 2$, split the middle term } \\ a(a - 7b) - 2b(a - 7b) \amp\quad \text{ Factor by grouping } \\ (a - 7b)(a - 2b) \amp\quad \text{ Our Solution } \end{align*}
    As the past few examples illustrate, it is very important to be aware of negatives as we find the pair of numbers we will use to split the middle term. Consier the following example, done incorrectly, ignoring negative signs.

    Warning 6.3.7.

    \begin{align*} x^2+5x-6 \amp \quad \text{ Want to multiply to $6$, add } 5 \\ x^2+2x+3x-6 \amp\quad \text{ $2$ and $3$, split the middle term } \\ x(x + 2) + 3(x - 2) \amp\quad \text{ Factor by grouping } \\ \text{???} \amp\quad \text{ Binomials do not match! } \end{align*}
    Because we did not use the negative sign with the six to find our pair of numbers, the binomials did not match and grouping was not able to work at the end. Now the problem will be done correctly.

    Example 6.3.8.

    \begin{align*} x^2+5x-6 \amp \quad \text{ Want to multiply to $-6$, add to } 5 \\ x^2+6x-x-6 \amp\quad \text{ $6$ and $-1$, split the middle term } \\ x(x+6)-1(x+6) \amp\quad \text{ Factor by grouping } \\ (x+6)(x-1)\amp\quad \text{ Our Solution } \end{align*}
    You may have noticed a shortcut for factoring these problems. Once we identify the two numbers that are used to split the middle term, these are the two num- bers in our factors! In the previous example, the numbers used to split the middle term were \(6\) and \(- 1\text{,}\) our factors turned out to be \((x + 6)(x - 1)\text{.}\) This pattern does not always work, so be careful getting in the habit of using it. We can use it however, when we have no number (technically we have a \(1\)) in front of \(x\) . In all \(2\) the problems we have factored in this lesson there is no number in front of \(x\text{.}\) If this is the case then we can use this shortcut. This is shown in the next few examples.

    Example 6.3.9.

    \begin{align*} x^2 - 7x - 18\amp \quad \text{ Want to multiply to $- 18$, add to } - 7 \\ \amp\quad \text{ $- 9$ and $2$, write the factors } \\ (x - 9)(x + 2) \amp\quad \text{ Our Solution } \end{align*}

    Example 6.3.10.

    \begin{align*} m^2 - m n - 30n^2 \amp \quad \text{ Want to multiply to $- 30$, add to } - 1 \\ \amp\quad \text{ $5$ and $- 6$, write the factors, don't forget second variable } \\ (m + 5n)(m - 6n) \amp\quad \text{ Our Solution } \end{align*}
    It is possible to have a problem that does not factor. If there is no combination of numbers that multiplies and adds to the correct numbers, then we say we cannot factor the polynomial, or we say the polynomial is prime. This is shown in the following example.

    Example 6.3.11.

    \begin{align*} x^2+2x+6 \amp \quad \text{ Want to multiply to $6$, add to } 2 \\ 1\times 6 \text{ and } 2 \times 3 \amp\quad \text{ Only possibilities to multiply to six, none add to } 2 \\ \text{ Prime, can't factor} \amp\quad \text{ Our Solution } \end{align*}
    When factoring it is important not to forget about the GCF. If all the terms in a problem have a common factor we will want to first factor out the GCF before we factor using any other method.

    Example 6.3.12.

    \begin{align*} 3x^2 -24x+45 \amp \quad \text{ GCF of all terms is $3$, factor this out } \\ 3(x^2-8x+15) \amp\quad \text{ Want to multiply to $15$, addt o } -8 \\ \amp\quad \text{ $- 5$ and $- 3$, write the factors } \\ 3(x-5)(x-3) \amp\quad \text{ Our Solution } \end{align*}
    Again it is important to comment on the shortcut of jumping right to the factors, this only works if there is no coefficient on \(x^2\) . In the next lesson we will look at how this process changes slightly when we have a number in front of \(x^2\text{.}\) Be careful not to use this shortcut on all factoring problems!
