    Section 5.7 Divide Polynomials

    Objective: Divide polynomials using long division.
    Dividing polynomials is a process very similar to long division of whole numbers. But before we look at that, we will first want to be able to master dividing a polynomial by a monomial. The way we do this is very similar to distributing, but the operation we distribute is the division, dividing each term by the monomial and reducing the resulting expression. This is shown in the following examples

    Example 5.7.1.

    \begin{align*} \frac{9x^5 + 6x^4 - 18x^3 - 24x^2}{3x^2} \amp \quad \text{ Divide each term in the numerator by } 3x^2 \\ \frac{9x^5}{3x^2} + \frac{6x^4}{3x^2} - \frac{18x^3}{3x^2} - \frac{24x^2}{3x^2} \amp\quad \text{ Reduce each fraction, subtracting exponents } \\ 3x^3+2x^2-6x-8 \amp\quad \text{ Our Solution } \end{align*}

    Example 5.7.2.

    \begin{align*} \frac{8x^3 +4x^2 -2x+6}{4x^2} \amp \quad \text{ Divide each term in the numerator by } 4x^2 \\ \frac{8x^3}{4x^2} + \frac{4x^2}{4x^2} - \frac{2x}{4x^2} + \frac{6}{4x^2} \amp\quad \text{ Reduce each fraction, subtracting exponents } \\ 2x+1-\frac{1}{2x}+\frac{3}{2x^2} \amp\quad \text{ Our Solution } \end{align*}
    The previous example illustrates that sometimes we will have fractions in our solution, as long as they are reduced this will be correct for our solution. Also interesting in this problem is the second term \(\frac{4x^2}{4x^2}\) divided out completely. Remember that this means the reduced answer is \(1\) not \(0\text{.}\)
    Long division is required when we divide by more than just a monomial. Long division with polynomials works very similar to long division with whole numbers.
    An example is given to review the (general) steps that are used with whole numbers that we will also use with polynomials.

    Example 5.7.3.

    \begin{align*} \Mydiv{4}{631} \amp \quad \text{ Divide front numbers: } \frac{6}{4} = 1\ldots \\ 1 \quad \amp\quad \text{ } \\ \Mydiv{4}{631} \amp\quad \text{ Multiply this number by divisor: } 1 \times 4 = 4 \\ \underline{-4 \quad} \amp\quad \text{ Change the sign of this number (make it subtract) and combine } \\ 23 \ \ \amp\quad \text{ Bring down next number } \\ \amp\quad \text{ } \\ 15 \quad \amp\quad \text{ Repeat, divide front numbers: } \frac{23}{4} = 5\ldots \\ \Mydiv{4}{631} \amp\quad \text{ } \\ \underline{-4 \quad} \amp\quad \text{ } \\ 23 \ \ \amp\quad \text{ Multiply this number by divisor: } 5 \times 4 = 20 \\ \underline{ -20\ \ } \amp\quad \text{ Change the sign of this number (make it subtract) and combine } \\ 31 \ \ \amp\quad \text{ Bring down next number } \\ \amp\quad \text{ } \\ 157 \amp\quad \text{ Repeat, divide front numbers: } \frac{31}{4}=7... \\ \Mydiv{4}{631} \amp\quad \text{ } \\ \underline{-4 \quad} \amp\quad \\ 23 \ \ \amp\quad \\ \underline{ -20 \ \ } \amp\quad \\ 31 \ \ \amp\quad \text{ Multiply this number by divisor: } 7 \times 4 = 28 \\ \underline{-28 \ \ } \amp\quad \text{ Change the sign of this number (make it subtract) and combine } \\ 3\quad \amp\quad \text{ We will write our remainder as a fraction, over the divisor, added to the end } \\ \amp\quad \text{ } \\ 157\frac{3}{4} \amp\quad \text{ Our Slution } \end{align*}
    This same process will be used to multiply polynomials. The only difference is we will replace the word “number” with the word “term”.
    Dividing Polynomials
    1. Divide front terms
    2. Multiply this term by the divisor
    3. Change the sign of the terms and combine
    4. Bring down the next term
    5. Repeat
    Step number 3 tends to be the one that students skip, not changing the signs of the terms would be equivalent to adding instead of subtracting on long division with whole numbers. Be sure not to miss this step! This process is illustrated in the following two examples.

    Example 5.7.4.

    \begin{align*} \frac{3x^3 -5x^2 -32x+7}{x-4} \amp \quad \text{ Rewrite problem as long division } \\ \\ \Mydiv{x-4}{3x^3 -5x^2 -32x+7} \amp\quad \text{ Divide front terms: } \frac{3x^3}{x}=3x^2 \\ \\ 3x^2 \qquad \qquad \qquad \quad \amp\quad \text{ } \\ \Mydiv{x-4}{3x^3 -5x^2 -32x+7} \amp\quad \text{ Multiply this term by divisor: } 3x^2(x-4)=3x^3-12x^2 \\ \underline{- 3x^3 + 12x^2\qquad \qquad } \amp\quad \text{ Change the signs and combine } \\ 7x^2 - 32x \qquad \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 3x^2 +7x\qquad \qquad \quad \amp\quad \text{ Repeat, divide front terms: } \frac{7x^2}{x}=7x \\ \Mydiv{x-4}{3x^3 -5x^2 -32x+7} \amp\quad \\ \underline{- 3x^3 + 12x^2\qquad \qquad} \amp\quad \\ 7x^2 - 32x \qquad \amp\quad \text{ Multiply this term by divisor: }7x(x - 4) = 7x^2 - 28x \\ \underline{-7x^2 + 28x \qquad } \amp\quad \text{ Change the signs and combine } \\ -4x+7 \ \ \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 3x^2 +7x-4\qquad \quad \amp\quad \text{ Repeat, divide front terms: } \frac{-4x}{x}=-4 \\ \Mydiv{x-4}{3x^3 -5x^2 -32x+7} \amp\quad \\ \underline{- 3x^3 + 12x^2\qquad \qquad} \amp\quad \\ 7x^2 - 32x \qquad \amp\quad \\ \underline{-7x^2 + 28x \qquad } \amp\quad \\ -4x+7\ \ \amp\quad \text{ Multiply this term by divisor: } -4(x-4)=-4x+16 \\ \underline{+4x-16} \amp\quad \text{ Change the signs and combine } \\ -9 \amp\quad \text{ Remainder put over divisor and subtracted (due to negative) } \\ 3x^2+7x-4-\frac{9}{x-4} \amp\quad \text{ Our solution } \end{align*}

    Example 5.7.5.

    \begin{align*} \frac{6x^3 -8x^2 +10x+103 }{2x+4 } \amp \quad \text{ Rewrite problem as long division } \\ \amp\quad \text{ } \\ \Mydiv{2x+4 }{ 6x^3 -8x^2 +10x+103} \amp\quad \text{Divide front terms: } \frac{6x^3}{2x}=3x^2 \\ \amp\quad \text{ } \\ 3x^2\qquad \qquad \qquad \qquad\amp\quad \text{ } \\ \Mydiv{2x+4 }{ 6x^3 -8x^2 +10x+103} \amp\quad \text{Multiply term by divisor: } 3x^2(2x + 4) = 6x^3 + 12x^2 \\ \underline{- 6x^3 - 12x^2 \qquad \qquad \quad} \amp\quad \text{ Change the signs and combine } \\ - 20x^2 + 10x \qquad \ \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 3x^2-10x \qquad \qquad \quad \amp\quad \text{ } \\ \Mydiv{2x+4 }{ 6x^3 -8x^2 +10x+103} \amp\quad \text{Repeat, divide front terms: } \frac{-20x^2}{2x}=-10x \\ \underline{- 6x^3 - 12x^2 \qquad \qquad \quad } \amp\quad \text{ Multiply this term by divisor: } \\ - 20x^2 + 10x\qquad \amp\quad -10x(2x+4)=-20x^2 -40x \\ \underline{+20x^2+40x \qquad \ } \amp\quad \text{ Change the signs and combine } \\ 50x + 103 \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 3x^2-10x+25 \qquad \quad \amp\quad \text{ } \\ \Mydiv{2x+4 }{ 6x^3 -8x^2 +10x+103} \amp\quad \text{Repeat, divide front terms: } \frac{50x}{2x}=25 \\ \underline{- 6x^3 - 12x^2 \qquad \qquad\quad } \amp\quad \\ - 20x^2 + 10x\qquad \ \amp\quad \\ \underline{+20x^2+40x \qquad} \amp\quad \\ 50x + 103 \amp\quad \text{ Multiply this term by divisor: } 25(2x + 4) = 50x + 100 \\ \underline{-50x-100} \amp\quad \text{ Change the signs and combine } \\ 3 \amp\quad \text{ Remainder is put over divsor and added (due to positive) } \\ \amp\quad \text{ } \\ 3x^2-10x+25+\frac{3}{2x+4} \amp\quad \text{ Our Solution } \end{align*}
    In both of the previous example the dividends had the exponents on our variable counting down, no exponent skipped, third power, second power, first power, zero power (remember \(x^0 = 1\) so there is no variable on zero power). This is very important in long division, the variables must count down and no exponent can be skipped. If they don’t count down we must put them in order. If an exponent is skipped we will have to add a term to the problem, with zero for its coefficient. This is demonstrated in the following example.

    Example 5.7.6.

    \begin{align*} \frac{ 2x^3 +42-4x}{x+3 } \amp \quad \text{ Reorder dividend, need $x^2$ term, add $0x^2$ for this } \\ \amp\quad \text{ } \\ \Mydiv{x+3 }{2x^3 +0x^2 -4x+42 } \amp\quad \text{ Divide front terms: } \frac{2x^3}{ x}=2x^2 \\ \amp\quad \text{ } \\ 2x^2\qquad \qquad \qquad \quad \amp\quad \text{ } \\ \Mydiv{x+3 }{2x^3 +0x^2 -4x+42 } \amp\quad \text{ Multiply this term by divisor: } 2x^2(x + 3) = 2x^3 + 6x^2 \\ \underline{-2x^3-6x^2\qquad\qquad \ } \amp\quad \text{ Change the signs and combine } \\ -6x^2 -4x \qquad \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 2x^2-6x \qquad \qquad \quad \amp\quad \text{ } \\ \Mydiv{x+3 }{2x^3 +0x^2 -4x+42 } \amp\quad \text{ Repeat, divide front terms: } \frac{-6x^2}{x}=-6x \\ \underline{-2x^3-6x^2\qquad\qquad \ } \amp\quad \\ -6x^2 -4x\qquad \amp\quad \text{ Multiply this term by divisor: } - 6x(x + 3) = - 6x^2 - 18x \\ \underline{+6x^2+18x\qquad } \amp\quad \text{ Change the signs and combine } \\ 14x+42 \amp\quad \text{ Bring down the next term } \\ \amp\quad \text{ } \\ 2x^2-6x+14 \qquad \amp\quad \text{ } \\ \Mydiv{x+3 }{2x^3 +0x^2 -4x+42 } \amp\quad \text{ Repeat, divide front terms: } \frac{14x}{x}=14 \\ \underline{-2x^3-6x^2\qquad\qquad\ } \amp\quad \\ -6x^2 -4x \qquad \amp\quad \\ \underline{+6x^2+18x\qquad } \amp\quad \\ 14x+42 \amp\quad \text{ Multiply this term by divisor: } 14(x + 3) = 14x + 42 \\ \underline{-14x-42} \amp\quad \text{ Change the signs and combine } \\ 0 \amp\quad \text{ No remainder } \\ \amp\quad \text{ } \\ 2x^2 -6x+14 \amp\quad \text{ Our Solution } \end{align*}
    It is important to take a moment to check each problem to verify that the exponents count down and no exponent is skipped. If so we will have to adjust the problem. Also, this final example illustrates, just as in regular long division, sometimes we have no remainder in a problem.
    World View Note: Paolo Ruffini was an Italian Mathematician of the early \(19\)th century. In \(1809\) he was the first to describe a process called synthetic division which could also be used to divide polynomials.
