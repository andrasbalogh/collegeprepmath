    Section 9.1 Solving with Radicals

    Objective: Solve equations with radicals and check for extraneous solutions.
    Here we look at equations that have roots in the problem. As you might expect, to clear a root we can raise both sides to an exponent. So to clear a square root we can rise both sides to the second power. To clear a cubed root we can raise both sides to a third power. There is one catch to solving a problem with roots in it, sometimes we end up with solutions that do not actually work in the equation. This will only happen if the index on the root is even, and it will not happen all the time. So for these problems it will be required that we check our answer in the original problem. If a value does not work it is called an extraneous solution and not included in the final solution.
    \begin{equation*} \textbf{ When solving a radical problem with an even index: check answers! } \end{equation*}

    Example 9.1.1.

    \begin{align*} \sqrt{7x+2}=4 \amp \quad \text{ Even index! We will have to check answers. } \\ \left(\sqrt{7x+2}\right)^2=4^2 \amp\quad \text{ Square both sides, simplify exponents } \\ 7x+2=16 \amp\quad \text{ Solve } \\ -2\ -2 \amp\quad \text{ Subtract $2$ from both sides } \\ \frac{7x}{7}=\frac{14}{7} \amp\quad \text{ Divide both sides by } 7 \\ x=2 \amp\quad \text{ Need to check answer in original problem } \\ \sqrt{7(2)+2}=4 \amp\quad \text{ Multiply } \\ \sqrt{14+2}=4 \amp\quad \text{ Add } \\ \sqrt{16}=4 \amp\quad \text{ Square root } \\ 4=4 \amp\quad \text{ True! It works! } \\ x=2 \amp\quad \text{ Our Solution } \end{align*}

    Example 9.1.2.

    \begin{align*} \sqrt[3]{x-1}=-4 \amp \quad \text{ Odd index, we don't need to check answer. } \\ \left(\sqrt[3]{x-1}\right)^3=\left(-4\right)^3 \amp\quad \text{ Cube both sides, simplify exponents } \\ x-1=-64 \amp\quad \text{ Solve } \\ +1\quad +1 \amp\quad \text{ Add $1$ to both sides } \\ x=-63 \amp\quad \text{ Our Solution } \end{align*}

    Example 9.1.3.

    \begin{align*} \sqrt[4]{3x+6}=-3 \amp \quad \text{ Even index! We will have to check answers } \\ \left(\sqrt[4]{3x+6}\right)^4=\left(-3\right)^4 \amp\quad \text{ Rise both sides to fourth power } \\ 3x+6=81\amp\quad \text{ Solve } \\ -6\ -6 \amp\quad \text{ Subtract $6$ from both sides } \\ \frac{3x}{3}=\frac{75}{3} \amp\quad \text{ Divide both sides by } 3 \\ x=25 \amp\quad \text{ Need to check answer in original problem } \\ \sqrt[4]{3(25)+6}=-3 \amp\quad \text{ Multiply } \\ \sqrt[4]{75+6}=-3 \amp\quad \text{ Add } \\ \sqrt[4]{81}=-3 \amp\quad \text{ Take root } \\ 3=-3 \amp\quad \text{ False, extraneous solution } \\ \text{No Solution} \amp\quad \text{ Our Solution } \end{align*}
    In this example one can also notice at the very beginning that in \(\sqrt[4]{3x+6}=-3\) the left hand side is always nonnegative, no mater what the value of \(x\) is, hence it will never be equal to the righ hand side \(-3\text{.}\)
    If the radical is not alone on one side of the equation we will have to solve for the radical before we raise it to an exponent

    Example 9.1.4.

    \begin{align*} x+\sqrt{4x+1}=5 \amp \quad \text{ Even index! We will have to check solutions } \\ -x\qquad \qquad -x \amp\quad \text{ Isolate radical by subtracting $x$ from both sides } \\ \sqrt{4x+1}=5-x \amp\quad \text{ Squarebothsides } \\ \left(\sqrt{4x+1}\right)^2=\left(5-x\right)^2 \amp\quad \text{ Evaluate exponents, recall } (a - b)^2 = a^2 - 2a b + b^2 \\ 4x+1=25-10x+x^2 \amp\quad \text{ Reorder terms } \\ 4x + 1 = x^2 - 10x + 25 \amp\quad \text{ Make equation equal zero } \\ -4x-1\qquad -4x-1 \amp\quad \text{ Subtract $4x$ and $1$ from both sides } \\ 0 = x^2 - 14x + 24 \amp\quad \text{ Factor } \\ 0=(x-12)(x-2) \amp\quad \text{ Set each factor equal to zero } \\ x-12=0 \text{ or } x-2=0 \amp\quad \text{ Solve each equation } \\ +12\ +12 \quad +2\ +2 \amp\quad \text{ } \\ x=12 \text{ or } x=2 \amp\quad \text{ Need to check answers in original problem } \\ \amp\quad \text{ } \\ (12)+\sqrt{4(12)+1} =5 \amp\quad \text{ Check $x=5$ first } \\ 12+\sqrt{48+1} =5 \amp\quad \text{ Add } \\ 12+\sqrt{49} =5 \amp\quad \text{ Take root } \\ 12+7=5 \amp\quad \text{ Add } \\ 19=5 \amp\quad \text{ False, extraneous root } \\ \amp\quad \text{ } \\ (2)+\sqrt{4(2)+1}=5 \amp\quad \text{ Check } x=2 \\ 2+\sqrt{8+1}=5 \amp\quad \text{ Add } \\ 2+\sqrt{9}=5 \amp\quad \text{ Take root } \\ 2+3=5\amp\quad \text{ Add } \\ 5=5 \amp\quad \text{ True! It works } \\ \amp\quad \text{ } \\ x=2 \amp\quad \text{ Our Solution } \end{align*}
    The above example illustrates that as we solve we could end up with an \(x^2\) term or a quadratic. In this case we remember to set the equation to zero and solve by factoring. We will have to check both solutions if the index in the problem was even. Sometimes both values work, sometimes only one, and sometimes neither works
    World View Note: The babylonians were the first known culture to solve quadratics in radicals - as early as \(2000\) BC!
    If there is more than one square root in a problem we will clear the roots one at a time. This means we must first isolate one of them before we square both sides.

    Example 9.1.5.

    \begin{align*} \sqrt{3x-8}-\sqrt{x}=0 \amp \quad \text{ Even index! We will have to check answers } \\ +\sqrt{x}\ +\sqrt{x} \amp\quad \text{ Isolate first root by adding $\sqrt{x}$ to both sides } \\ \sqrt{3x-8}=\sqrt{x} \amp\quad \text{ Square both sides } \\ \left( \sqrt{3x-8}\right)^2=\left(\sqrt{x}\right)^2 \amp\quad \text{ Evaluate exponents } \\ 3x-8=x \amp\quad \text{ Solve } \\ -3x\quad -3x \amp\quad \text{ Subtract $3x$ from both sides } \\ -8=-2x \amp\quad \text{ Divide both sides by } -2 \\ \frac{-8}{-2}=\frac{-2x}{-2} \amp\quad \text{ } \\ 4=x \amp\quad \text{ Need to check answer in original } \\ \sqrt{3(4)-8}-\sqrt{4}=0 \amp\quad \text{ Multiply } \\ \sqrt{12-8}-\sqrt{4}=0 \amp\quad \text{ Subtract } \\ \sqrt{4}-\sqrt{4}=0\amp\quad \text{ Take roots } \\ 2-2=0 \amp\quad \text{ Subtract } \\ 0 = 0\amp\quad \text{ True! It works } \\ x = 4 \amp\quad \text{ Our Solution } \end{align*}
    When there is more than one square root in the problem, after isolating one root and squaring both sides we may still have a root remaining in the problem. In this case we will again isolate the term with the second root and square both sides. When isolating, we will isolate the term with the square root. This means the square root can be multiplied by a number after isolating.

    Example 9.1.6.

    \begin{align*} \sqrt{2x+1}-\sqrt{x}=1 \amp \quad \text{ Even index! We will have to check answers } \\ +\sqrt{x}\ +\sqrt{x} \amp\quad \text{ Isolate first root by adding $\sqrt{x}$ to both sides } \\ \sqrt{2x+1}=\sqrt{x} +1 \amp\quad \text{ Square both sides } \\ \left(\sqrt{2x+1}\right)^2=\left(\sqrt{x} +1\right)^2 \amp \quad \text{ Evaluate exponents, recall } (a + b)^2 = a^2 + 2a b + b^2 \\ 2x+1=x+2\sqrt{x}+1 \amp\quad \text{ Isolate the term with the root } \\ -x-1\quad \qquad -x-1 \amp\quad \text{ Subtract $x$ and $1$ from both sides } \\ x=2\sqrt{x} \amp \quad \text{ Square both sides } \\ \left(x \right)^2=\left(2\sqrt{x}\right)^2 \amp\quad \text{ Evaluate exponents } \\ x^2=4x \amp\quad \text{ Make equation equal zero } \\ -4x\ -4x \amp \quad \text{ Subtract $x$ from both sides } \\ x^2-4x=0 \amp\quad \text{ Factor } \\ x(x-4)=0 \amp\quad \text{ Set each factor equal to zero } \\ x=0 \text{ or } x-4=0 \amp \quad \text{ Solve } \\ +4\ +4 \amp\quad \text{ Add $4$ to both sides of second equation } \\ x=0 \text{ or } x=4 \amp\quad \text{ Need to check answers in original } \\ \amp \quad \text{ } \\ \sqrt{2(0)+1}-\sqrt{(0)}=1 \amp\quad \text{ Check $x=0$ first } \\ \sqrt{1}-\sqrt{0}=1\amp\quad \text{ Take roots } \\ 1-0=1 \amp \quad \text{ Subtract } \\ 1=1 \amp\quad \text{ True! It works } \\ \amp\quad \text{ } \\ \sqrt{2(4)+1}-\sqrt{(4)}=1 \amp \quad \text{ Check } x=4 \\ \sqrt{8+1}-\sqrt{4}=1 \amp\quad \text{ Add } \\ \sqrt{9}-\sqrt{4}=1 \amp\quad \text{Take roots} \\ 3-2=1 \amp\quad \text{ Subtract} \\ 1=1 \amp\quad \text{True! It works } \\ x=0 \text{ or } 4 \amp\quad \text{ Our Solution } \end{align*}

    Example 9.1.7.

    \begin{align*} \sqrt{3x+9}-\sqrt{x+4}=-1 \amp \quad \text{ Even index! We will have to check answers } \\ +\sqrt{x+4}\ +\sqrt{x+4} \amp\quad \text{ Isolate the first root by adding } \sqrt{x+4 } \\ \sqrt{3x+9}=\sqrt{x+4}-1 \amp\quad \text{ Square both sides } \\ \left(\sqrt{3x+9}\right)^2=\left(\sqrt{x+4}-1\right)^2 \amp \quad \text{ Evaluate exponents } \\ 3x+9=x+4-2\sqrt{x+4}+1 \amp\quad \text{ Combine like terms } \\ 3x+9=x+5-2\sqrt{x+4} \amp\quad \text{ Isolate the term with radical } \\ -x-5\ -x-5 \qquad \qquad \amp \quad \text{ Subtract $x$ and $5$ from both sides } \\ 2x+4=-2 \sqrt{x+4} \amp\quad \text{ Square both sides } \\ \left( 2x+4 \right)^2=\left(-2 \sqrt{x+4} \right)^2 \amp\quad \text{ Evaluate exponents } \\ 4x^2 +16x+16=4(x+4) \amp \quad \text{ Distribute } \\ 4x^2 +16x+16=4x+16 \amp\quad \text{ Make equation equal zero } \\ -4x-16\ -4x-16 \amp\quad \text{ Subtract $4x$ and $16$ from both sides } \\ 4x^2 +12x=0\amp \quad \text{ Factor } \\ 4x(x+3)=0 \amp\quad \text{ Set each factor equal to zero } \\ 4x=0 \text{ or } x+3=0 \amp\quad \text{ Solve } \\ \div 4\ \div 4\qquad -3\ -3 \amp \quad \text{ } \\ x=0 \text{ or } x=-3 \amp\quad \text{ Check solutions in original } \\ \amp\quad \text{ } \\ \sqrt{3(0)+9}-\sqrt{(0)+4}=-1 \amp \quad \text{ Check $x=0$ first } \\ \sqrt{9}-\sqrt{4}=-1 \amp\quad \text{ Take roots } \\ 3-2=-1 \amp\quad \text{ Subtract } \\ 1=-1 \amp \quad \text{ False, extraneous solution } \\ \amp\quad \text{ } \\ \sqrt{3(-3)+9}-\sqrt{(-3)+4}=-1 \amp\quad \text{ Check } x=-3 \\ \sqrt{-9+9}-\sqrt{-3+4}=-1 \amp \quad \text{ Add } \\ \sqrt{0}-\sqrt{1}=-1 \amp\quad \text{ Take roots } \\ 0-1=-1 \amp\quad \text{ Subtract } \\ -1=-1 \amp \quad \text{ True! It works } \\ \amp\quad \text{ } \\ x=-3 \amp\quad \text{ Our Solution } \end{align*}
