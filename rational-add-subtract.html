    Section 7.4 Add and Subtract

    Objective: Add and subtract rational expressions with and without common denominators.
    Adding and subtracting rational expressions is identical to adding and subtracting with integers. Recall that when adding with a common denominator we add the numerators and keep the denominator. This is the same process used with rational expressions. Remember to reduce, if possible, your final answer.

    Example 7.4.1.

    \begin{align*} \frac{x-4}{x^2-2x-8} + \frac{x+8}{x^2-2x-8} \amp \quad \text{Same denominator, add numerators, combine like terms } \\ \frac{2x+4}{x^2-2x-8} \amp\quad \text{Factor numerator and denominator } \\ \frac{2(x+2)}{(x+2)(x-4)} \amp\quad \text{Divide out } (x+2) \\ \frac{2}{x-4} \amp\quad \text{ Our Solution } \end{align*}
    Subtraction with common denominator follows the same pattern, though the subtraction can cause problems if we are not careful with it. To avoid sign errors we will first distribute the subtraction through the numerator. Then we can treat I like an addition problem. This process is the same as “add the opposite” we saw when subtracting with negatives.

    Example 7.4.2.

    \begin{align*} \frac{6x-12}{3x-6}-\frac{15x-6}{3x-6} \amp \quad \text{ Add the opposite of the second fraction (distribute negative) } \\ \frac{6x-12}{3x-6}+\frac{-15x+6}{3x-6} \amp\quad \text{ Add numerators, combine like terms } \\ \frac{-9x-6}{3x-6} \amp\quad \text{ Factor numerator and denominator } \\ \frac{-3(3x+12)}{3(x-2)} \amp\quad \text{Divide out common factor of } 3 \\ \frac{-(3x+2)}{(x-2)} \amp\quad \text{ Our solution } \end{align*}
    World View Note: The Rhind papyrus of Egypt from \(1650\) BC gives some of the earliest known symbols for addition and subtraction, a pair of legs walking in the direction one reads for addition, and a pair of legs walking in the opposite direction for subtraction.
    When we don’t have a common denominator, we will have to find the least common denominator (LCD) and build up each fraction so the denominators match. The following example shows this process with integers.

    Example 7.4.3.

    \begin{align*} \frac{5}{6}+\frac{1}{4} \amp \quad \text{ The LCD is $12$. Build up, multiply $6$ by $2$ and $4$ by } 3 \\ \left(\frac{2}{2}\right)\frac{5}{6}+\frac{1}{4}\left(\frac{3}{3}\right) \amp\quad \text{ Multiply } \\ \frac{10}{12}+\frac{3}{12} \amp\quad \text{ Add numerators } \\ \frac{13}{12} \amp\quad \text{ Our solution } \end{align*}
    The same process is used with variables.

    Example 7.4.4.

    \begin{align*} \frac{7a}{3a^2b}+\frac{4b}{6ab^4} \amp \quad \text{ The LCD is $6a^2b^4$. We will then build up each fraction } \\ \left(\frac{2b^3}{2b^3}\right)\frac{7a}{3a^2b}+\frac{4b}{6ab^4}\left(\frac{a}{a}\right) \amp\quad \text{Multiply first fraction by $2b^3$ and second by } a \\ \frac{14ab^3}{6a^2b^4}+\frac{4ab}{6a^2b^4} \amp\quad \text{ Add numerators, no like terms to combine } \\ \frac{14ab^3+4ab}{6a^2b^4} \amp\quad \text{Factor numerator } \\ \frac{2ab(7b^3+2)}{6a^2b^4} \amp\quad \text{ Reduce, dividing out factors $2$, $a$, and } b \\ \frac{7b^3+2}{3ab^3} \amp\quad \text{ Our solution } \end{align*}
    The same process can be used for subtraction, we will simply add the first step of adding the opposite.

    Example 7.4.5.

    \begin{align*} \frac{4}{5a}-\frac{7b}{4a^2} \amp \quad \text{ Add the opposite } \\ \frac{4}{5a}+\frac{-7b}{4a^2} \amp\quad \text{LCD is $20a^2$. Build up denominators } \\ \left(\frac{4a}{4a}\right)\frac{4}{5a}+\frac{-7b}{4a^2}\left(\frac{5}{5}\right) \amp\quad \text{Multiply first fraction by $4a$, second by } 5 \\ \frac{16a-35b}{20a^2} \amp\quad \text{ Our solution } \end{align*}
    If our denominators have more than one term in them, we will need to factor first to find the LCD. Then we build up each denominator using the factors that are missing on each fraction.

    Example 7.4.6.

    \begin{align*} \frac{6}{8a+4}+\frac{3a}{8} \amp \quad \text{Factor denominators to find LCD } \\ 4(2a+1) \ \ 8 \amp\quad \text{LCD is $8(2a+1)$, build up each fraction } \\ \left(\frac{2}{2}\right)\frac{6}{4(2a+1)}+\frac{3a}{8}\left(\frac{2a+1}{2a+1}\right) \amp\quad \text{Multiply first fraction by $2$, second by } 2a+1 \\ \frac{12}{8(2a+1)}+\frac{6a^2+3a}{8(2a+1)} \amp\quad \text{Add numerators} \\ \frac{6a^2+3a+12}{8(2a+1)} \amp\quad \text{Our solution } \end{align*}
    With subtraction remember to add the opposite.

    Example 7.4.7.

    \begin{align*} \frac{x+1}{x-4}-\frac{x+1}{x^2-7x+12} \amp \quad \text{Add the opposite (distribute negative) } \\ \frac{x+1}{x-4}+\frac{-x-1}{x^2-7x+12} \amp\quad \text{Factor denominators to find LCD } \\ x-4 \ \ (x-4)(x-3) \amp\quad \text{LCD is $(x-4)(x-3)$, build up each fraction } \\ \left(\frac{x-3}{x-3}\right)\frac{x+1}{x-4}+\frac{-x-1}{x^2-7x+12} \amp\quad \text{Only first fraction needs to be multiplied by } x-3 \\ \frac{x^2-2x-3}{(x-3)(x-4)}+\frac{-x-1}{(x-3)(x-4)} \amp\quad \text{Add numerators, combine like terms } \\ \frac{x^2-3x-4}{(x-3)(x-4)} \amp\quad \text{ Factor numerator } \\ \frac{(x-4)(x+1)}{(x-3)(x-4)} \amp\quad \text{ Divide our $x-4$ factor } \\ \frac{x+1}{x-3} \amp\quad \text{ Our solution } \end{align*}
